我之前说过：「我还年轻，我渴望上路。带着最初的激情，追寻着最初的梦想，感受着最初的体验，我们上路吧。 ——凯鲁亚克《在路上》。 「他妈的，我老了！」 ——MyersGuo」。   

今天，我想起来研究生毕业那会儿，好多同学奔着银行、运营商、老师、公务员、国企去，也有同学奔着BAT走。但首选的仍然是银行、运营商。跟我同实验室的同学，「王鹏」去了河南郑州的中国移动，入职的职位竟是HR，把我震惊到了，但他拿着稳定的收入，不愁房子（老家有房），也买了车子，活的逍遥自在。另一个实验室好友，「王国海」去了广州的一家专利审查公司，出了死工资，还有提成——审查通过一个专利提成将近800左右，在广州收入也不少，买了车子和房子（广州现今的放假1.5万,2017.05.04），活的也是逍遥自在。我的室友刘方伟同学去了浙江下面一个地级市的银行，工资比较低，不知道现在活的是不是也逍遥自在。   

毕业时的我，怀着「闯一闯」的心踏上了北上之路，刚来的时候喝着西北风，感觉激情荡漾的。近4年过去了，和同学聊天，在广州、河南的同学「逍遥自在」着，我眼睁睁的看着北京的房价像放飞的风筝飞走，继续喝着西北的风和不时的霾。如果让我重新选择，我可能会去选择公务员吧，做到像《人民的名义》里面的大贪官，或者为什么服务啥的。这个工作还高大上。   

小时候看各种电视剧，就有各种梦想。看到律师，相当律师，看到医生，觉得做个主刀医生也不错。看到警察，想着警察也很威风。看到比尔盖茨，想着计算机原来搞不好能成首富。。。。   

后来发现事实是酱紫的：  

```
#include <stdio.h>
int main(){
    printf("hello,world");
}
```

有人还会问为你[「如何检查输入的IP是否合法」](https://myersguo.github.io/2017/04/18/check-ip.html)，以及二叉树的横向排列，全排列算法，如何设计一个朋友圈的问题。当你看到你的 code run 起来的时候，你感到这或许就是梦想。而你拖着匹配的身躯回到家，跟老婆说，我今天好累。你可能又有点怀疑人生。   

雷军说，小米为发烧而生，继而说，小米，让每个人享受到科技的乐趣。  
王传福说，以厂为家、爱厂如家，继而说，技术为王，创新为本，改善环境，实现人类的可持续发展。  

MyersGuo说，向左走，向右走，追寻内心，开心就好。
