昨夜写完YII WHISPER，我就想，YII那么多实用的类，不好入门啊，能不能裁剪一下，弄个简单的MVC框架？   

晚饭回来，这个想法又激发起来。让我们动起收来。   

假如我的入口文件是hello.php ,那我想要定义一个控制器类，控制入口:  

```
class controller{
    public function run()
    {
    
    }
}
$app = new controller();
$app->run();
```

这样，我们的程序的骨架就起来了，下面就开始弄具体的实现。URL如何路由：  

```
public function run()
{
    $routeName = $this->parseUrl();
    $route = new $routeName();
    $data = $route->run();
    echo $this->render('views/'.$routeName.'.php', $data);
}
```

通过解析URL，获取到路由器，执行路由器的类，然后把路由器执行的结果渲染到模版。   


解析URL，我们先用最精简的


```
public function parseUrl()
{   
    $uri = $_SERVER['REQUEST_URI']; 
    $uri = trim($uri,'.php');
    if(($pos=strpos($uri,'?'))!==false)
            $uri=substr($uri,0,$pos);
    if($uri[0]==='/')
            $uri=substr($uri,1);
    return $uri;
}
  
```
我们的入口是hello.php,解析的URL后路由就是:`hello`,找到的view就是views\hello.php,我们写一个路由的类：  

```
class hello{
    public function run(){
        $ret = array(
            'errno'=>0,
            'errmsg'=>'ok',
            'data'=>array('name'=>'myersguo'),
        );                                                                                                                    
        return $ret;
    }   
}
```

这样整个流程就变成了：

```
class hello{
    public function run(){
        $ret = array(
            'errno'=>0,
            'errmsg'=>'ok',
            'data'=>array('name'=>'myersguo'),
        );                                                                                                                    
        return $ret;
    }   
}
class controller{
    public function parseUrl()
    {   
        $uri = $_SERVER['REQUEST_URI']; 
        $uri = trim($uri,'.php');
        if(($pos=strpos($uri,'?'))!==false)
                $uri=substr($uri,0,$pos);
        if($uri[0]==='/')
                $uri=substr($uri,1);
        return $uri;

    }   
    public function render($file, $data)
    {   
        extract($data);
        ob_start();
        ob_implicit_flush(false);
        require($file);
        return ob_get_clean();    
    }   
    public function run()
    {   
        $routeName = $this->parseUrl();
        $route = new $routeName();
        $data = $route->run();
        echo $this->render('views/'.$routeName.'.php',$data);
    }   
}

$app = new controller();
$app->run();

``

视图文件`views\hello.php`写成：

```
<div>hello,<?php echo $data['name'];?></div> 
```

这样，我们执行hello.php就输出了hello,myersguo
