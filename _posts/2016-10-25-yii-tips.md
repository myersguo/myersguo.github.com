### Yii(YiiBase) ###

它是应用的基础类，提供类的查找方法(包括组件的创建）等。

```
spl_autoload_register(array('YiiBase','autoload'));
require(YII_PATH.'/base/interfaces.php');
```


#### getPathOfAlias #### 

目的：生成`路径别名`    



```
检查是否已经导入

if(isset(self::$_aliases[$alias]))
    return self::$_aliases;

根目录
$pos = strpos($alias, '.');
$rootAlias = substr($alias,0,$pos);
获取路径
self::$_aliases[$alias] = self::$_aliases[$rootAlias]. DIRECTORY_SEPARATOR. str_replace('.', DIRECTORY_SEPARATOR, substr($alias, $pos+1));

```

#### import ####

目的：生成`classmap`


放置重复导入


```
#已导入过
if(isset(self::$_imports[$alias]))
    return self::$_imports[$alias];
#已定义过
if(class_exists($alias, false) || interface_exists($alias, false))
    return self::$_imports[$alias] = $alias;

#包含namespace\\
$namespace = str_replace('\\', '.', ltrim(substr($alias,0,$pos),'\\'));
获取namespace的路径
$path = self::getPathOfAlias($namespace);
#找到类
$classFile = $path.DIRECTORY_SEPARATOR.substr($alias,$pos+1).'.php';
加入classmap
self::$classMap[$alias]=$classFile;

# 以alias形式导入
#是否导入类
$className=(string)substr($alias,$pos+1);
$isClass=$className!=='*';
#类已存在
if($isClass && (class_exists($className,false) || interface_exists($className,false)))
    return self::$_imports[$alias]=$className;



```




#### createComponet ####

根据组件参数，创建组件对象并导入.  



```
$type = $config['class'];
$object = new $type;
foreach($config as $key => $value)
    $object->$key = $value;

```

```
Yii::createComponent(
```
